# Roleplay
É uma API para gerenciamento de mesas de RPG aquele jogo em que pessoas  se reúnem uma mesa para interpretar personagens e jogar dados para tomar decisões dentro do jogo.

A ideia é que as pessoas interessadas em jogar RPG encontrem uma mesa e solicitem participar dela.
O mestre será responsável por cadastrar as mesas e aceitar os jogadores interessados desenvolvendo este
aplicativo.
